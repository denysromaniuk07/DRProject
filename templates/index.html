Project Description

Purpose of the application: Extract and analyze product reviews from Ceneo.pl.

Description of the application's operation: The user enters a product ID, and the system retrieves all available reviews, generates statistics, and allows exporting data.

Libraries used: Flask, BeautifulSoup, Requests, Matplotlib, Pandas.

Applied approaches: Object-Oriented Programming (OOP), Web Scraping, Data Visualization.
